# 🏠 Nemo Store Insight - 서울 강남권 상가 데이터 분석

서울 강남권(양재, 도곡, 개포 등) 상업용 부동산 시장의 심층 데이터 분석 및 비즈니스 통찰을 제공하는 프로젝트입니다.

## 🚀 프로젝트 개요
- **목적**: 네모(Nemo) 플랫폼의 매물 데이터를 활용하여 상권 특성을 분석하고 창업 전략을 도출합니다.
- **분석 대상**: 서울 강남구/서초구 일대 상가 매물 약 470여 건
- **주요 기능**: 
  - 상권별 보증금/월세/권리금 분포 분석
  - 업종별 가격 상관관계 및 입지 전략 도출
  - 핵심 키워드(TF-IDF) 분석을 통한 시장 트렌드 파악

## 📊 대시보드 확인하기
이 프로젝트의 분석 결과는 아래 링크에서 인터랙티브 대시보드로 확인하실 수 있습니다.

## 📂 폴더 구조
- `src/`: 데이터 수집 및 분석 스크립트
- `data/`: 분석에 사용된 원천 데이터 (DB 파일 등은 .gitignore 처리)
- `images/`: 분석 결과 시각화 차트 이미지
- `docs/`: 프로젝트 관련 문서
- `index.html`: 메인 대시보드 웹 페이지

## 🛠 기술 스택
- **언어**: Python 3.x
- **데이터 분석**: Pandas, Matplotlib, Seaborn
- **웹 프론트엔드**: HTML5, Tailwind CSS, Lucide Icons
